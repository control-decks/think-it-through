# Make The Call

- State the decision being made and the criteria that matter most.
- Choose one direction clearly; do not hide behind an unranked list or “it depends.”
- Give the shortest rationale that makes the choice defensible.
- Name the decisive tradeoff and mention a rejected alternative only when it clarifies the choice.
- State the assumptions, confidence level, and the signal that would justify reversing the decision.

Make the decision with the available context. Do not create an execution plan unless the user also requests map-it-out.
