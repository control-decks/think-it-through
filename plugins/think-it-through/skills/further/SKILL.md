# 🚀 Think Further

**ID:** `think-it-through/further`\
**HACP:** `0.4`\
**Kind:** `operation`\
**Mode:** `transform`\
**Traits:** `read-only`, `semantic`, `creative-extension`\
**Default Binding:** Current Working Object, then current thought\
**Accepts:** `hacp/content`, `hacp/result`\
**Requires:** `hacp/extendable-thought`\
**Produces:** `think-it-through/extended-thought`\
**Duration:** `once`

**Effect:** Find the strongest supported edge of the bound object and extend it
by one non-obvious implication, combination, or possibility.

**Limits:** Do not scatter into broad ideation, invent evidence, restate the
object, choose a direction, recommend actions, or continue beyond one leap.

## Resolve the leap

1. Recover the object's claims, evidence, constraints, and unresolved edge.
2. Select one extension that is both non-obvious and materially connected to
   that edge.
3. Preserve established claims. Mark the new part as extrapolation whenever it
   depends on an unverified assumption.
4. Stop after the one useful leap.

## Format

Begin the complete combo trace with
`> 🎯 **<binding>** → 🚀 **FURTHER**` when used alone. Add
`→ 🚀 **FURTHER**` after earlier cards in a combo.

State the extension directly. Add one short `Extrapolation` note only when the
new step depends on an unsupported assumption.
