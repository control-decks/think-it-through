---

# Map It Out

- Derive the goal and accepted direction from context instead of reopening settled choices.
- Define the smallest useful scope, including meaningful exclusions.
- Order concrete steps by dependency and outcome.
- Include owners, timing, or checkpoints only when they affect execution.
- State how success will be validated and distinguish blockers from details that can be decided later.

Produce the smallest plan that another person or agent could execute safely. Ask one question only if a missing decision truly blocks the plan. Do not execute the plan.
